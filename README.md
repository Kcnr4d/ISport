## Temat projektu
Produktem jest aplikacja mobilna **ISport**. System jest przeznaczony dla osób chcących uprawiać sport bez konieczności zakupu drogiego sprzętu oraz dla wypożyczalni, które chcą usprawnić swoje codzienne funkcjonowanie.

## Wizja systemu
Głównym celem systemu jest optymalizacja zarządzania wypożyczalnią poprzez automatyzację procesów. Kluczowe filary systemu to:
* **Sprawne zarządzanie:** Przyspieszenie obsługi wypożyczeń i zwrotów oraz minimalizacja błędów przy wydawaniu sprzętu.
* **Pełna kontrola:** Monitorowanie dostępności zasobów w czasie rzeczywistym oraz wgląd w historię sprzętu.
* **Bezpieczeństwo i trwałość:** Systematyczne planowanie serwisów technicznych, co przedłuża żywotność sprzętu i zwiększa bezpieczeństwo.
* **Automatyzacja komunikacji:** Wysyłanie automatycznych powiadomień i przypomnień o terminach zwrotu, co redukuje opóźnienia.
